
# Smart Home Automation System (Microcontroller Project)

A microcontroller-based smart home automation system developed using embedded C.  
The project controls and monitors multiple sensors and modules such as temperature sensor (LM35), RFID module, motion sensor, and keypad.

## Features
- Temperature monitoring using LM35 sensor
- RFID-based access control
- Motion detection
- Keypad-controlled operations
- LCD display output
- Modular embedded C architecture
